To run these demos, you need an Atari Falcon 030. Some STE demos can work on the Falcon as well. Most of the earlier demos should work on the C-Lab Falcon, Falcon clones and accelerated Falcons. However some of the more recent demos, particularly those that use the DSP for calculations, often use timed routines that might not work on accelerated systems. Note also that most of these demos are designed on systems with RGB monitor displays, hence many demos are RGB only. Finally, many demos like to make full use of the 4 megabytes of standard Falcon memory! This last problem can be solved by using a configurable loader program like God-Load.
