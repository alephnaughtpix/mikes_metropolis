downloading tools

Why do you need all these tools?

All of the demos and diskmags for many different computer formats featured in these pages are stored in UNIX file-systems on FTP servers, and can thus be accessed by any internet ready computer. Most of them use standard archiving tools like Arc, Lharc, Zip, Zoo and the newer Arj and LZX, which exist for all computers from 16-bit upwards. In addition, there are often specific programs with associated file formats for each platform. Demos on both Amiga and Atari 16-bit platforms were often done in raw sector data format, so there are dedicated programs on those platform to convert a raw disk image into a file, and back again. There are also binary convertors to allow 8-bit file formats (eg early disk formats, audio tape!!) to be transferred to more modern file formats. Most of these file formats originated from formats used by 16-bit emulators of 8-bit machines (eg Spectrum "snapshots"), but are now also used as standard file format transfer of 8-bit software on the Internet.
