A NEW DISK MAGAZINE FOR BUDDING FALCON DEMO CODERS.

How to Code is a shareware disk magazine aimed at people interested in programming for fun on the Atari Falcon 030, in particular programming for the demo scene. If you ever wanted to do some of the amazing effects shown in the latest Falcon demos, but didn't know where to start, then HTC aims to provide the answer. Articles covers such areas as new demo effects, optimisation routines, information on file formats, or beginners courses in coding. The magazine is compiled and edited by members of EKO, the group behind award-winning demos such as "Are You Experienced?" and "EKO System".

HTC is now in it's third revision, and includes articles on:

  • Fire effects (As seen in many demos on Falcon, Amiga, PC)
  • Mars mountains (As seen in the "Are You Experienced" demo by EKO)
  • Graphics techniques used in Doom.
  • Running Linux and X-Windows on an Atari Falcon.
  • How to decode GIF pictures.
  • Beginners courses in C and the Falcon DSP.

All of the programming articles include detailed source code examples.

The magazine uses a smooth and user-friendly article displayer using the Falcons unique hardware capabilities, and also includes an alternative GEM friendly article displayer, which can be used in conjuction with Devpac or any other GEM-based language editor as a reference guide whilst programming.
