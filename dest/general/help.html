How to get around Michael's sprawling mess of pages.


If you've got this far, then presumably you know that you can go to different pages, or different parts of pages by selecting highlighted bits of text or graphics. There are also common icons, or bits of small graphics, which you can select to quickly move around my pages. If you are not viewing thing this on text-only viewers, you might have also noticed little graphics icons called qbullets.

There are two types of pages here: menu pages and ordinary pages. Ordinary pages are generally of the following format:

Title
| |
Main Text
| |
Icon Strip

I'm sure you know what title is, and you can probably take a fair guess at the meaning of the phrase main text, but perhaps the icon strip idea might need a bit of explaining...

What's an "Icon strip"?

Well, as the name suggests, it's basically a strip of selectable icons. Each of them allow you to move to a different pages in my amazing sprawling empire! So let's look at what each of the icons do:


    Go to previous page.

    This icon takes you to the previous page in the series. Of course if there is no previous page, then this icon won't appear!


    Go to local menu page.

    This icon takes you to the local menu page- in other words, the last menu page you were in.


    Go to next page.

    This icon takes you to the next page in the series. Of course if there is no next page, then this icon won't appear!


    Go to my Home Page.

    This icon takes you to a lovely little page which tells you all about the amazing person behind this sprawling mess!


    Go to Main Index.

    This icon takes you straight to the main index of my amazing sprawling empire. Quite handy if you get lost!


    Mail me.

    This icon allows you to email to the amazing person behind this sprawling mess. (Note- This doesn't work with Mosaic!)


What about Menu Pages?

The format of menu pages is slighty more complex, but not much:

Title
| |
Link to Quick menu
| |
Long description of menu options
| |
Quick Menu
| |
Icon strip

There's also a couple of new icons as well....

    Latest Updates

    Selecting this will allow you to view information on the latest updates to my pages. You usually see this icon appearing on menu pages.


    Help page.

    Well I wonder if you can guess what this does! Yes, that's right- it brings you to this page! That must have been really hard.


    Quick Menu

    This allows you to bypass the long menu description, and go straight to a list of menu choices.

What are those little "Qbullets" icons about?

Qbullets are little pieces of graphics stuck next to links, denoting the type of link. There is a more detailed explanation of Qbullets here, courtesy of our local WebSoc , but here is a quick reminder of what they mean in the context of my pages. (Apologies to text-only viewers- skip this bit.)

= List/Index

= New feature

= Link to another server

= Link to FTP server

= Download file

= Link to a picture

= Help/Explantion page

= Email to...

= Fill-out form

= cgi-script

= guess!

The following refer to the size of a downloaded file/picture...
= file < 80K
= file 80-200K
= file 200-700K
= file 700-1000K
= file 1000K+

What If I see this on a page?

This means that I am currently working on the page at the moment! Be careful- one or two things might not work, so try reloading it every couple of minutes if this happens.
