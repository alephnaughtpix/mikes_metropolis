What's this about a "GIF Tax"?

Hey, all you people who use GIFs! (Most of the Internet, as it happens.) Read this! Compuserve and Unisys have reached an agreement over the use of the LZW algorithm (patented by Unisys) in GIF pics, which effectively mean that any program using the GIF format which is sold for profit has to be registered and pay money to use the GIF format. (It doesn't include PD or past products, so WWW servers like this won't be affected, although if developers stop using the GIF format, they might be...) This also could affect TIFF formats using the LZW algorithm.
