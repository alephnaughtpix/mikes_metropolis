To run these demos, you need an Atari STFM. Most of these demos work on STEs as well, unless indicated. One or two even work to limited extent on Falcon 030s. Most of these demos will work on a half megabyte system, but 1 meg is usually preferred. You'll see what I mean if you have a half meg system, and you have to keep reloading the selection menus!!

The 3.5K Megademo (STNCC2 3.5K demos compilation):

A "megademo", compiled by me, of the eight screens entered an ST News Coding Conference 2 competition. The object of this competition was to do as good a demo as possible in 3.5 kilobytes. These were done before 4K demo competitions became fashionable, so it's also a bit of history!

VT52 Demo:

Originally released on an issue of Maggie, this is a demo in a text file !!! The file uses VT52 codes to draw a cube and add a scroller with rhythmic "gamelean" bell music. Run this demo in low res, by simply double-clicking on it and displaying it as you would any normal text file. Do it from an HD or ramdisk though, because if you try if from a floppy it'll pause every so often to read the file from disk, and it'll look like a PC demo! (NOTE- Almost unbelievably, this text file demo is not Falcon compatible!!!!! The colour codes used in this demo do not correspond to the colour codes in TOS 4.x!!!!)

By: Synergy (19 Aug 1996)

    SNRGYTXT.LZH (45 K)
